## ಶೀರ್ಷಿಕೆ: ನೀವು ಏನು ಕಲಿಯುವಿರಿ

- Scratch ಸಂಖ್ಯೆಗಳನ್ನು ಹೋಲಿಸಲು ಆಪರೇಟರ್‌ಗಳನ್ನು ಬಳಸಿ
- Scratch ನ್ನಲಿ sprite ಬಣ್ಣವನ್ನು ಮುಟ್ಟಿದಾಗ ಕಂಡುಹಿಡಿಯಲು code ಸೇರಿಸಿ
- Scratch ನಲ್ಲಿ ಸಮಯವನ್ನು ದಾಖಲಿಸಲು ವೇರಿಯೇಬಲ್ ಬಳಸಿ
